（一）Vue-安装

2017-06-10 • 分类: Vue

步骤有些简洁，算是傻瓜式安装，当然啦，管它黑马还是白马，能够跑起来就好咧。

vue安装步骤：

1. 使用淘宝镜像安装 cnpm

  • git 命令执行 $ npm install -g cnpm –registry=https://registry.npm.taobao.org
  • 看 cnpm 是否可用 命令执行 cnpm -v

2. 使用 cnpm 全局（-g）安装 vue-cli

  • $ cnpm install -g vue-cli
  • $ vue

3. 创建一个基于 “webpack” 模板的新项目

  • $ vue init webpack my-first-vue-project

4. 安装依赖

  • $ cd my-project-name(项目名称) //到此目录下
  • $ npm install

5. 运行环境

  • $ npm run dev （文件目录下运行）

6. sublime 支持 vue 高亮 （使用sublime开发工具的同学安装啦）

  • install Vue Syntax Highlight / jade

7. vue更改默认端口号8080为指定端口

  • config -> index.js -> dev: port: 指定端口
